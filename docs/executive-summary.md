Overview

The Excel Project Control System is a lightweight, Excel-based solution designed
to help small construction and service businesses manage projects, invoices,
payments, expenses, change orders, and performance KPIs in a centralized and
structured way.

The solution applies business analysis best practices to transform fragmented
tracking (multiple spreadsheets, emails, and manual follow-ups) into a coherent,
decision-support system.

Business Problem

Small teams often struggle with:

unclear project status and progress

missed invoice follow-ups

limited visibility into cash flow

difficulty tracking profitability across multiple projects

These issues increase administrative effort and reduce the quality of
decision-making.

Solution Summary

The system:

centralizes project, financial, and operational data

enforces data consistency through validation rules

separates data entry from reporting

provides real-time portfolio KPIs through a dashboard

Business Analysis Approach

The solution was designed using a structured BA approach:

Use Case Diagram to identify user interactions

Future-state Process Flows to clarify responsibilities

High-level Data Model to define core entities and relationships

Use Case Specifications & User Stories to translate needs into requirements

Traceability to ensure alignment between requirements and implementation

Key Benefits

Improved visibility into cash flow and outstanding receivables

Better control over scope and contract value changes

Faster, data-driven decision-making

Reduced manual errors and rework

Intended Audience

Office Administrators
